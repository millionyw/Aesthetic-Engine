# .ai_context

## 项目摘要
构建一个本地运行的基于 CLIP 的个人审美偏好引擎，用少量标注实现大规模图片筛选。

## 技术栈约定
- Python 3.10+
- PyTorch（必须适配 MPS/Mac 或 CUDA/Nvidia 加速）
- HuggingFace Transformers（CLIP 模型）
- 数据存储格式：Pickle/JSON

## 架构决策
选择 Linear Probing 而非 Fine-tuning：冻结大模型骨干，仅训练末端小模型，以降低样本需求、算力成本并提升隐私场景的可控性。
